<ch03>
normalization	正規化，データを0-1に収まるようにする
whitening	白色化，データの分布を均一にする

batch		バッチ，複数のデータを1つの配列として扱える
	※データ転送がボトルネックになるときバッチを使うと処理を短縮できる
	※大きな配列を効率よく処理できる最適化がされてるからデータ読み込みに対して演算の割合を多くするだけ速くなる<ch04>
パーセプトロンの収束定理	線形分離可能であれば有限回の学習で解ける

NN・DNN←人の介入を避けパターンを見つける	=特徴量までも機械が学習する
特徴量	データから本質的なデータを抽出する変換器

なぜ認識精度ではなく損失関数を使うのか
=予測を連続値で出す(離散値も一回連続で出してる)のでそれも考慮したいから

Numerical Differentiation(数値微分)	微小な差分によって微分を求める		=パワー微分
Analytic Differentiation(解析的な微分)	数式の展開によって微分を求める		=真の微分


勾配が示す方向 = 関数の値を最も減らす方向
Gradient Method(勾配法)：勾配方向に進むことで関数の値を減らす
SGD：Stochastic Gradient Descent(確率的勾配降下法)，ミニバッチを無作為に取る

エポック：訓練データを全て使い切る回数	Ex) 訓練データ10000，ミニバッチ100→エポック100


<ch05>
Computational Graph(計算グラフ)：計算の過程をグラフで表したもの，局所的な計算，途中を保持できる
	Forward Propagation(順伝播)
		1．計算グラフを構築する
		2．計算グラフを左から右に進める

	Backward Propagation(逆伝播)

ReLU(Rectified Linear Unit)

Affineレイヤ：順伝播の行列の積
	Affine		行列の積

数値微分	近似値，O(N)	=N回の逆伝播が必要
誤差逆伝播法	解析的，O(1)	=1回の逆伝播でできる


<ch06>
重み←均一だとだめ，各層のニューロンを増やす意味がなくなる
アクティベーション：活性化関数後の出力

Batch Normalization：角層ののアクティベーションの分布を調整する
	学習係数を大きくできる		=学習を速くできる
	初期値にロバスト		=初期値にそれほど依存しない
	過学習を抑制する		=Drop outの必要性を減らす

過学習
	表現力が高すぎる
	訓練データが少ない

訓練データ	パラメータの学習
検証データ	ハイパーパラメータの性能を評価
テストデータ	汎化性能をチェック

ハイパーパラメータの最適化：いい値が存在する範囲を絞り込んでい(10のべき乗単位で) Ex) 10^-3 ~ 10^3
→グリッドサーチの規則的な探索よりもランダムサーチの方が良い結果になる


<ch07>
Fully Connected(全結合)：隣接する層のニューロン全てが連結している	Ex) Affineレイヤ
※畳み込み層は全結合でない，フィルタ内の領域としか連結されない

Feature Map(特徴マップ)：畳み込み層の入出力データ

・Convolution Layer(畳み込み層)：畳み込み演算をする層，チャネルの和
	Padding(パディング)	入力データの周囲をデータを埋める，出力サイズを調整する
	Stride(ストライド)	フィルターを適応する間隔，フィルターを適応する
・Pooling Layer(プーリング層)：縦横方向の空間を小さくする層，位置変化にロバスト，チャネルに独立

im2col(image to column)：データを行列に展開する	※Numpyは行列計算なら高速でできる

edge(エッジ)：変化する境界
blob(ブロブ)：塊のある領域

層を重ねるごとに抽出される情報は抽象化されていく	=反応する対象が高度になる

LeNet		Convolution層とPooling層でできた初期のCNN，Pooling層は間引くだけ
AlexNet		LeNetの進化版，MaxPooling・局所正規化・Dropout

<ch08>
Data Augmentation(データ拡張)

層を深くする利点：
	少ないパラメータ数で受容野を広くカバーできる
		受容野	ニューロンの変化を生じさせる局所的な空間領域
	学習データを少なくできる(パラメータが少ないから)
	階層的に分解できる		=Devide and Conquer

VGG		Convolution層を連続でしてからPooling層でサイズを半分(交互にやらない)
GoogLeNet	横方向にも深さをもたせる(インセプション構造)，サイズ1のフィルターでチャネル数を減らす
ResNet		スキップ構造で層を深くしてもいいようにした
		↑逆伝播で勾配消失を防げる

・学習のボトルネックは積和演算
・ビット数を削減する高速化
Deep Learningにはロバスト性があるのでビット数は16bit(half float，半浮動小数点)で十分
	BNN	Binarized Neural Networks，重みや中間データを1bitで表現する

物体認識		MNIST
物体検出		画像中からクラスと位置を特定する

R-CNN：オブジェクトらしい領域を抽出してからクラス分類する
	1．Input image
	2. Extract region proposals	Selective Searchで似ているピクセルを結合
	3. Compute CNN features
	4. Classify regions

セグメンテーション	ピクセルレベルでクラス分類する

FCN：Fully CNN，畳み込み層だけで構成	=空間情報を保持したまま処理できる
	・全結合層を畳み込み層に置き換える
	・1回のForward処理で全てのピクセルをクラス分類
	・Deconvolution layer(逆畳み込み層)で拡大処理

画像キャプション

NIC：Neural Image Caption，CNNで得た特徴をRNNに渡す
	マルチモーダル処理	複数の種類の情報を組み合わせる	Ex) 画像と自然言語

画像スタイル変換：中間データをコンテンツ画像の中間データに近づくようにする
	スタイル画像		画風
	コンテンツ画像		元の画像

画像生成：無から画像を生成する
	DCGAN	Deep Convolutional Generative adversarial networks(深層畳み込み敵対性生成ネットワーク)
		Generator(生成者)とDiscriminator(識別者)を競い合わせて精巧な生成と鑑定をつくる

DQN：Deep Q-Network(強化学習)，試行過程から自律的に学習する，入力データが画面だけでいい
	最適行動価値係数	見込みの報酬，CNNで関数を近似する
